Resize to Fit

Github (Source Code) Metroidvania M32 Game Jam Play Game

Game Format

  • Genre: Endless
  • Published: 15 June 2026
  • Type: Single Player
  • Platform: PC / Web

Technical Details

Unity 3D C#

Game Mechanics

  • Movement: Player can move the Cylinder in horizontal direction
  • Wall (Obstacle) Generation: Procedural generation of oncoming walls, each featuring gap(s) of varying widths and positions.
  • Resizing: Player can resize the Cylinder's diameter (shrink or grow) to fit through the gap
  • Score: The larger the size of the cylinder, the more points earned based on the Scoring Formula
  • Speed Progression: Speed gradually increases after every successfully passed gap, reducing reaction time.

Scoring Formula

slope = (maxPoints - minPoints) / (maxScale - minScale)
points = minPoints + ((currentScale - minScale) * slope)

Dynamics

  • Risk vs Reward Balancing: Players constantly decide between going bigger for higher scores or shrinking to survive tighter gaps.
  • Last-second saves Tight situations where players barely shrink or reposition in time to slip through a gap.
  • Overambition: Players often push for higher scores on risky gaps, leading to avoidable crashes when they overcommit.

Core Loop

Read gap between approachin walls -> Adjust Size and Position -> Pass Safely -> Gain score -> Speed increases [Next Gap] -> Repeat

Goal

  • 1. Survive as long as possible by fitting through gaps between endless approaching walls
  • 2. Maximize your score by growing as large as possible while still fitting through the gap

Rules

  • Collision: Any contact with a wall results in immediate game over.
  • Fit: The cylinder must fit fully within the gap to pass through it.
  • Reward: Larger sizes give higher scores, while the cylinder can be shrunk to any size to fit the gap.

Feedback

  • Speed increases when passing through the gap
  • Audio feedback plays while increasing the cylinder size
  • Audio feedback plays when player hits a wall
  • UI feedback displays the score dynamically when passing through a gap
